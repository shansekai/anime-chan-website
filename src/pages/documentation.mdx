## Introduction

**Base url:** https://animechanapi.xyz/api/

The base url contains information about all available API's resources.
All requests are `GET` requests and go over `https`. All responses will return data in `json`.

_Sample request_

```
https://animechanapi.xyz/api/
```

```json
{
  "quotes": "https://animechanapi.xyz/api/quotes",
  "random": "https://animechanapi.xyz/api/quotes/random",
  "by_anime": "https://animechanapi.xyz/api/quotes?anime=naruto",
  "by_character": "https://animechanapi.xyz/api/quotes?char=madara uchiha"
}
```

### Quotes

Returns 10 quotes by default. Pagination is implemented. To retreeve the next set of 10 quotes add `/?page=2`

```
https://animechanapi.xyz/api/quotes
```

```json
{
  "status": "ok",
  "statusCode": 200,
  "data": {
    "quote": "People, who can’t throw something important away, can never hope to change anything.",
    "character": "Armin Arlert",
    "anime": "Shingeki no Kyojin"
  }
}
```

### Random

Returns one single random quote.

```
https://animechanapi.xyz/api/quotes/random
```

```json
{
  "status": "ok",
  "statusCode": 200,
  "data": {
    "quote": "People, who can’t throw something important away, can never hope to change anything.",
    "character": "Armin Arlert",
    "anime": "Shingeki no Kyojin"
  }
}
```
